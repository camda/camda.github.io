# 漫感

## 关于漫感

拖了许久，原本想自己写个博客，无奈每次都被页面的各种设计搞得烦死，所以博客的事情就慢慢拖着。

hexo，jekyll这些静态blog系统以前也接触了一下，但是感觉缺少blog的很多机制，比如说评论这些，作为一个后台开发者有点不能容忍这种事的发生所以也就没有下文。

直到最近，发现越来越多的东西需要有个地方记录下来，所以想了半天打算花个一两天时间使用jekyll搭建一个静态blog，当做一个暂时的blog。

这个blog基于jekyll，使用配置也很简单，没有接触过的朋友可以取[jekyll](http://jekyllrb.com)的网站了解下，我呢，在基于[Freud Kang](http://www.hifreud.com)主题的基础上做了一些修改（后期还会做改动，不过大致的样子是不会变的，只会优化细节），形成自己风格的blog。

这个blog只是暂时作为blog使用，后续想着如果没有比较好看（符合我审美学）的blog系统，打算自己开发一个，这个到时会开源出来和大家一起分享。当然，现在这个blog我也会后续继续优化的。
